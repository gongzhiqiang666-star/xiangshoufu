
1.协议规则
 
传输方式	 
采用HTTP传输
 
提交方式	 
采用POST方法提交
 
数据格式	 
提交和返回数据都为json
 
字符编码	 
统一采用UTF-8字符编码
 
签名算法	 
RSA
 
签名/加密要求	 
详见签名说明
 
异步通知返回参数	 
success
